# いしかわ外国語案内メーカー

## 📝 概要（Overview）

このリポジトリは、**飲食店の事業者がインバウンド観光客向けの案内文を簡単に作成し、印刷できる Webアプリ**です。

- 日本語で入力チェックボックスするだけで、店頭掲示用の案内文を生成
- 印刷を前提としたシンプルなレイアウト
- ブラウザだけで動作する軽量なツールを目指しています

※言語対応：現状は日本語の案内文生成を想定しています。

---

## ✨ 主な機能（Features）

- フォーム入力からの案内文テンプレート生成
- 印刷前に内容を確認できるプレビュー画面
- A4 に合わせた印刷用レイアウト
